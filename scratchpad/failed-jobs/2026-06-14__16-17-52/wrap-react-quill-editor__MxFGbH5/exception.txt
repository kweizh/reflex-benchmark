# Wrap React-Quill as a Reflex Custom Component

## Background
Reflex lets developers wrap arbitrary React/npm components into native-feeling Python components. Your job is to wrap the [`react-quill`](https://www.npmjs.com/package/react-quill) rich-text editor (version `2.0.0`) as a Reflex component and use it on the index page of a Reflex app. The editor must be bound to a `str` state variable, and the raw HTML it produces must be displayed live below the editor.

## Requirements
- Define a Python class `ReactQuill` that subclasses `NoSSRComponent` (importable from `reflex.components.component`).
  - `library` must reference `react-quill` pinned to version `2.0.0` (e.g. `"react-quill@2.0.0"`).
  - `tag` must be `"ReactQuill"`.
  - Declare a prop `value: rx.Var[str]`.
  - Declare an event trigger `on_change: rx.EventHandler[lambda v: [v]]` (the lambda must return the value as the first positional argument).
  - Pull in Quill's CSS so the editor renders correctly. You may use `lib_dependencies = ["quill@2.0.0"]` together with an explicit CSS reference, or include the Snow theme stylesheet via the `rxconfig.py` `stylesheets=[...]` setting (e.g. `https://cdn.quilljs.com/1.3.6/quill.snow.css`).
  - Export a module-level factory `react_quill = ReactQuill.create`.
- Define a Reflex `State` with:
  - A base var `content: str = ""`.
  - An event handler `set_content(self, v: str)` that assigns `self.content = v`.
- Build a page mounted at `/` that:
  - Renders the wrapped `react_quill(value=State.content, on_change=State.set_content)` editor.
  - Renders the raw HTML that the editor produces below it using `rx.html(State.content)` so the live preview reflects edits.
- Use `uv` to manage the Python environment exactly as described in the implementation hints.

## Implementation Hints
- Initialize the project with `uv init` + `uv add reflex` + `uv run reflex init --template blank` in the project directory.
- Put the wrapper class in its own Python module (e.g. `react_quill.py`) inside the Reflex app package, and import the `react_quill` factory from your main app module.
- Because the editor accesses `window`/`document`, wrap it via `NoSSRComponent` so Reflex emits a dynamic import.
- To trigger the npm install for the frontend, run `uv run reflex export --frontend-only --no-zip`. This populates `.web/package.json` with the `react-quill` dependency declared in `library`.
- Keep all event-trigger lambdas serializable: returning `[v]` is required so Reflex forwards the new HTML string to the event handler.
- The editor's `on_change` callback receives the HTML string. Storing it as `content: str` and rendering it through `rx.html` is enough to mirror the editor below itself.
