# Reflex Async Paginated User Directory ## Background Reflex is a full-stack Python framework that compiles Python UI definitions into a Next.js/React frontend driven by a FastAPI/websocket backend. State is held in `rx.State` subclasses, and database models are declared with `rx.Model` (a SQLModel wrapper). Long-running database queries should be executed in background events (`@rx.event(background=True)`) using the async session helper `rx.asession()`, while UI mutations are guarded by `async with self:` to acquire the exclusive state lock. Your task is to implement a Reflex application that serves a paginated user directory backed by SQLite.
