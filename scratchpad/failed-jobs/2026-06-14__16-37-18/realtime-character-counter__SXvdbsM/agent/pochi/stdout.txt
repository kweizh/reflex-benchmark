In this task you will build a small reactive page using Reflex that demonstrates a `rx.text_area` bound to a State var, plus two derived counters implemented as cached computed vars. ## Requirements * Build a single-page Reflex app that renders at the index route `/`. * The page MUST contain: * A `rx.text_area` component with `id="content_input"` that is bound to a State var named `content`.
